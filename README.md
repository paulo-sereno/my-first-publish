# Meu Primeiro Projeto

## Descrição
Um projeto simples utilizando as ferramentas essenciais de desenvolvimento: Terminal, VS Code e GitHub.

## Tecnologias
- **VS Code**: Editor de código
- **Terminal**: Gerenciamento de comandos
- **Git/GitHub**: Controle de versão

## Como Começar
1. Clonar o repositório
    ```bash
    git clone <url-do-repositorio>
    ```
2. Abrir no VS Code
3. Fazer alterações e commits
4. Fazer push para o GitHub
